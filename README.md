 Bank Customer Purchase Prediction

 Project Overview
This project uses a **Decision Tree Classifier** to predict whether a customer will purchase a product/service based on demographic and behavioral data from the UCI Bank Marketing dataset.

 Features Used
- **Demographics:** Age, Job, Marital Status, Education
- **Behavioral/Financial:** Balance, Housing Loan, Personal Loan, Active Campaign Contacts
